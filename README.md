# 📧 Email Extractor Automation Project

## 📌 Description
This project is a simple Python automation script that reads a text file and extracts all email addresses using regular expressions.

---

## ⚙️ Features
- Reads data from a text file
- Extracts email addresses using regex
- Removes duplicate emails
- Saves results into a new file
- Displays summary in terminal

---

## 🧠 Concepts Used
- Python File Handling
- Regular Expressions (re module)
- Sets (for removing duplicates)
- Basic Automation scripting
